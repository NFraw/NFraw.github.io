VSCode快捷键

本文最后更新于 2024年11月3日 中午

VSCode快捷键(英文版PDF可跳转)

常用快捷键

  • 显示命令面板: Ctrl+Shift+P, F1
  • 快速打开/跳转到文件: Ctrl+P
  • 新建窗口/实例: Ctrl+Shift+N
  • 关闭窗口/实例: Ctrl+Shift+W
  • 用户设置: Ctrl+,
  • 键盘快捷键: Ctrl+K Ctrl+S

基本编辑

  • 剪切行 (空选择): Ctrl+X
  • 复制行 (空选择): Ctrl+C
  • 向上/下移动行: Alt+↑ / ↓
  • 向上/下复制行: Shift+Alt + ↓ / ↑
  • 删除行: Ctrl+Shift+K
  • 在下方插入行: Ctrl+Enter
  • 在上方插入行: Ctrl+Shift+Enter
  • 跳转到匹配括号: Ctrl+Shift+\
  • 增加/减少行缩进: Ctrl+] / [
  • 跳转到行首/尾: Home / End
  • 跳转到文件开始: Ctrl+Home
  • 跳转到文件结束: Ctrl+End
  • 向上/下滚动行: Ctrl+↑ / ↓
  • 向上/下滚动页: Alt+PgUp / PgDn
  • 折叠/展开区域: Ctrl+Shift+[ / ]
  • 折叠/展开所有子区域: Ctrl+K Ctrl+[ / ]
  • 折叠/展开所有区域: Ctrl+K Ctrl+0 / Ctrl+K Ctrl+J
  • 添加行注释: Ctrl+K Ctrl+C
  • 移除行注释: Ctrl+K Ctrl+U
  • 切换行注释: Ctrl+/
  • 切换块注释: Shift+Alt+A
  • 切换自动换行: Alt+Z

导航

  • 显示所有符号: Ctrl+T
  • 跳转到行…: Ctrl+G
  • 跳转到文件…: Ctrl+P
  • 跳转到符号…: Ctrl+Shift+O
  • 显示问题面板: Ctrl+Shift+M
  • 跳转到下一个错误或警告: F8
  • 跳转到上一个错误或警告: Shift+F8
  • 导航编辑器组历史记录: Ctrl+Shift+Tab
  • 后退/前进: Alt+ ← / →

搜索和替换

  • 查找: Ctrl+F
  • 替换: Ctrl+H
  • 查找下一个/上一个: F3 / Shift+F3
  • 选择所有出现的匹配项: Alt+Enter
  • 将选择添加到下一个查找匹配项: Ctrl+D
  • 将最后一个选择移动到下一个查找匹配项: Ctrl+K Ctrl+D
  • 切换区分大小写/正则表达式/全字匹配: Alt+C / R / W

多光标和选择

  • 插入光标: Alt+Click
  • 在上方/下方插入光标: Ctrl+Alt+ ↑ / ↓
  • 撤销上次光标操作: Ctrl+U
  • 在每行选择的末尾插入光标: Shift+Alt+I
  • 选择当前行: Ctrl+L
  • 选择当前选择的所有出现: Ctrl+Shift+L
  • 选择当前单词的所有出现: Ctrl+F2
  • 扩展选择: Shift+Alt+→
  • 缩小选择: Shift+Alt+←
  • 列 (框) 选择: Ctrl+Shift+Alt + (箭头键)
  • 列 (框) 选择向上/下翻页: Ctrl+Shift+Alt +PgUp/PgDn

丰富语言编辑

  • 触发建议: Ctrl+Space, Ctrl+I
  • 触发参数提示: Ctrl+Shift+Space
  • 格式化文档: Shift+Alt+F
  • 格式化选择: Ctrl+K Ctrl+F
  • 跳转到定义: F12
  • 快速查看定义: Alt+F12
  • 在侧边打开定义: Ctrl+K F12
  • 快速修复: Ctrl+.
  • 显示引用: Shift+F12
  • 重命名符号: F2
  • 删除尾随空格: Ctrl+K M
  • 更改文件语言: Ctrl+K Ctrl+X

编辑器管理

  • 关闭编辑器: Ctrl+F4, Ctrl+W
  • 关闭文件夹: Ctrl+K F
  • 拆分编辑器: Ctrl+ 1 / 2 / 3
  • 聚焦到第 1/2/3 个编辑器组: Ctrl+K Ctrl+ ←/→
  • 聚焦到上一个/下一个编辑器组: Ctrl+Shift+PgUp / PgDn
  • 将编辑器左/右移动: Ctrl+K ← / →

文件管理

  • 新建文件: Ctrl+N
  • 打开文件…: Ctrl+O
  • 保存: Ctrl+S
  • 另存为…: Ctrl+Shift+S
  • 保存所有: Ctrl+K S
  • 关闭: Ctrl+F4
  • 关闭所有: Ctrl+K Ctrl+W
  • 重新打开关闭的编辑器: Ctrl+Shift+T
  • 保持预览模式编辑器打开: Ctrl+K Enter
  • 打开下一个: Ctrl+Tab
  • 打开上一个: Ctrl+Shift+Tab
  • 复制活动文件路径: Ctrl+K P
  • 在资源管理器中显示活动文件: Ctrl+K R
  • 在新窗口/实例中显示活动文件: Ctrl+K O

显示

  • 切换全屏: F11
  • 切换编辑器布局 (水平/垂直): Shift+Alt+0
  • 放大/缩小: Ctrl+ = / -
  • 切换侧边栏可见性: Ctrl+B
  • 显示资源管理器/切换焦点: Ctrl+Shift+E
  • 显示搜索: Ctrl+Shift+F
  • 显示源控制: Ctrl+Shift+G
  • 显示调试: Ctrl+Shift+D
  • 显示扩展: Ctrl+Shift+X
  • 在文件中替换: Ctrl+Shift+H
  • 切换搜索详细信息: Ctrl+Shift+J
  • 显示输出面板: Ctrl+Shift+U
  • 打开 Markdown 预览: Ctrl+Shift+V
  • 在侧边打开 Markdown 预览: Ctrl+K V
  • 禅模式 (Esc Esc 退出): Ctrl+K Z

调试

  • 切换断点: F9
  • 开始/继续: F5
  • 停止: Shift+F5
  • 单步进入/单步跳出: F11 / Shift+F11
  • 单步跳过: F10

集成终端

  • 显示集成终端: Ctrl+`
  • 创建新终端: Ctrl+Shift+`
  • 复制选择: Ctrl+C
  • 粘贴到活动终端: Ctrl+V
  • 向上/下滚动: Ctrl+↑ / ↓
  • 向上/下滚动页: Shift+PgUp / PgDn
  • 滚动到顶部/底部: Ctrl+Home / End
